# ✅ ИСПРАВЛЕНИЕ ПЕРЕСЧЁТА ПРИ СМЕНЕ ГРУППЫ

**Дата:** 10.03.2026  
**Статус:** ✅ **ИСПРАВЛЕНО**

---

## 🐛 ПРОБЛЕМА

При смене группы участника (например, с VIP 300₽ на Обычную 150₽) **не пересчитывались**:
- `paid_until_month` (дата до которой оплачено)
- `balance` (баланс/аванс)
- `next_payment` (следующий платёж)

**Пример проблемы:**
- Алимов Д. внёс 150₽
- Группа: VIP (300₽/мес)
- `balance`: 150₽ (аванс)
- `paid_until_month`: NULL (не оплачено ни одного месяца)

**При смене на Обычную (150₽):**
- **Ожидалось:** `balance=0₽`, `paid_until_month=2026-04`
- **Было:** Данные не менялись

---

## 🔧 ПРИЧИНА

В функции `update_participant` в `routers/participants.py` **не вызывалась** функция пересчёта `recalculate_participant_fields` при смене группы.

---

## ✅ РЕШЕНИЕ

### 1. Добавлен вызов `recalculate_participant_fields` при смене группы

**Файл:** `backend/routers/participants.py`

```python
if participant.group_id is not None and participant.group_id != p.group_id:
    # ... код смены группы ...
    
    p.group_id = participant.group_id
    
    # 🆕 Пересчитываем поля участника после смены группы
    recalculate_participant_fields(db, p)
```

### 2. Переписана функция `recalculate_participant_fields`

**Файл:** `backend/services/participant_service.py`

**Новая логика:**
1. Складываем **ВСЕ платежи** участника в общую сумму
2. Делим на **НОВЫЙ monthly_fee** группы
3. Отсчитываем месяцы от `start_date`
4. Остаток записываем в `balance`

**Пример расчёта:**
```python
total_amount = 150₽  # Все платежи
monthly_fee = 150₽   # Новая группа

months_paid = total_amount / monthly_fee = 150 / 150 = 1
balance = total_amount % monthly_fee = 150 % 150 = 0₽
paid_until_month = start_date + months_paid = 2026-03 + 1 = 2026-04
```

## 🚀 ЧТО ИЗМЕНИЛОСЬ ДЛЯ ПОЛЬЗОВАТЕЛЯ

Теперь при смене группы участника **автоматически пересчитываются**:

1. **Оплачено до (paid_until_month)** — на основе новой стоимости группы
2. **Баланс (balance)** — остаток от деления на новую стоимость
3. **Следующий платёж** — вычисляется из `paid_until_month`

**Пример:**
- Была группа VIP (300₽), внесено 150₽ → `balance=150₽`, `paid_until=NULL`
- Сменили на Обычную (150₽) → **автоматически:** `balance=0₽`, `paid_until=2026-04`
